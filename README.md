# 个人知识库

个人知识管理与持续学习系统，用于记录日常思考、构建主题知识体系、追踪学习进度。

## 项目说明

### 定位
- **性质**：个人知识管理系统（PKM）
- **目标**：将碎片化思考结构化，形成可复用的知识资产
- **风格**：专业术语归纳，去口语化

### 核心原则
1. **记录优先**：先写下来，再整理归类
2. **主题驱动**：以领域为单位组织知识，而非时间
3. **渐进式积累**：从 Inbox → Daily → Topics 逐步沉淀
4. **定期回顾**：每周/月回顾，保持知识库活力

## 目录结构

```
Personal/
├── INDEX.md              ← 全局导航，快速定位
├── README.md             ← 项目说明与使用指南
├── Daily/                ← 每日笔记，按日期命名
├── Inbox/                ← 收件箱，随手记的想法
├── Topics/               ← 主题知识体系
│   ├── 金融/             ← 金融知识体系（17章）
│   └── 个人形象/         ← 个人形象管理
└── Templates/            ← 笔记模板
```

## 工作流

### 日常记录
1. 想法先丢 `Inbox/`，不纠结格式
2. 每日在 `Daily/` 写总结，关联当日主题
3. 定期将 Inbox 内容归档到对应 Topic

### 主题积累
1. 在 `Topics/` 下维护各领域 README.md
2. 重要笔记以独立文件存放，README 中用双链引用
3. 每个 Topic 末尾维护"相关笔记"索引

### 定期维护
- **每周**：清空 Inbox，回顾 Daily 笔记
- **每月**：更新 Topic 结构，补充遗漏内容
- **每季**：审视知识体系，调整方向
